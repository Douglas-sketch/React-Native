# 📱 Meu Primeiro App React Native

Este repositório foi criado para meus estudos de **React Native com Expo**, utilizando o **GitHub Codespaces** como ambiente principal de desenvolvimento.

---

## 🚀 Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- GitHub Codespaces
- Git e GitHub

---

## 🎯 Objetivo do projeto

O objetivo deste projeto é aprender os fundamentos do React Native na prática, criando telas, componentes, botões, estilos, estados e futuramente recursos mais avançados de aplicativos mobile.

Este repositório servirá como meu espaço de aprendizado, testes e evolução no desenvolvimento de apps.

---

## 📚 O que estou estudando

Neste projeto, estou aprendendo:

- Estrutura básica de um app React Native
- Componentes como `View`, `Text`, `Pressable`, `TextInput`, `Image` e `ScrollView`
- Uso de `useState`
- Eventos com `onPress`
- Estilização com `StyleSheet`
- Organização de arquivos
- Testes usando Expo
- Versionamento com Git e GitHub

---

## 🧠 Primeiro exemplo do app

O primeiro app criado é um contador simples, onde o usuário pode clicar em um botão para aumentar o número exibido na tela.

Esse exemplo ajuda a entender conceitos importantes como:

- Estado
- Funções
- Eventos
- Renderização
- Estilos
